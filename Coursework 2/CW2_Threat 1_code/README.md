# CW2 Threat 1 Defense

This repository contains a Python-based proof-of-concept security middleware for mitigating TCP SYN flood and related denial-of-service behavior through sliding-window rate limiting and IP blacklisting.

## File
- `neobank_security_middleware.py`: main middleware prototype

## Features
- Sliding-window rate limiting
- Automatic IP blacklisting
- Lightweight Python implementation
