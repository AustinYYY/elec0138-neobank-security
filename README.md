# ELEC0138 Security and Privacy Coursework Repository

This repository contains the code, supporting files, and demonstration materials for our **ELEC0138 Security and Privacy** coursework project.

The project is organised into two main parts:

- **Coursework 1** – Threat Modelling and Attack Simulation  
- **Coursework 2** – Security and Privacy Defence Strategy  

This repository is intended to provide public access to the project code and materials for coursework submission, demonstration, and review.
