# CW2 Threat 1 Attack Simulation

This repository contains a Python-based simulation of a TCP SYN flood attack for NeoBank coursework 2.

## File
- `syn_flood_simulation.py`: main simulation script

## Output Figures
- `half_open_connections.png`
- `backlog_usage.png`
- `legitimate_success_rate.png`
- `latency_estimate.png`

## Purpose
This simulation demonstrates the effect of a TCP SYN flood attack on server backlog usage, connection success rate, and latency, as well as the impact of mitigation mechanisms such as SYN cookies and filtering.
